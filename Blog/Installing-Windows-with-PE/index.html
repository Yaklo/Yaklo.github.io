准备

在开始前请先浏览完整篇教程，并在 准备 章节下载准备完需要的内容、软件
必要的内容 > 在U盘做好WePE，下载系统镜像，下载系统激活软件
可选的内容 > 杀毒软件、压缩解压软件、聊天软件等，自行搜索下载

制作PE：请至少准备一个8GB的U盘（是USB3.0接口更好）
制作PE：准备一台能联网的电脑（有USB3.0接口更好）
系统安装：老爷机除外的电脑
系统安装：推荐使用带USB3.0及以上（蓝、红口）的主板和U盘
如果不是3.0及以上的也没关系，只是速度会慢一些

推荐使用32GB或以上U盘，方便日常使用以及存放更多的Win或其他系统的镜像~
教程以 Windows10 及 Windows11 为基准，Windows7 官方镜像没有中文加上系统老旧故不做教程，请自行研究~

U盘安装微PE

前往微PE官网 > 微PE工具箱
点击右上角 下载 > 下载最新微PE工具箱 > 查看下载地址列表
会显示一个赞助页面，点已捐赠未捐赠都可以显示下载链接，非强制
推荐下载64位
教程编撰时为V2.2，如果有更新可下载最新，软件基本都长一样不用担心

打开下载好的WePE安装程序，文件名大概长这样WePE64_V2.2.exe
①点击右下角 其他安装方式 的 U盘图标
②选择 待写入U盘 ,按 盘符 选择，不知道盘符的话打开 文件资源管理器 查看就是 此电脑、我的电脑 啦
③请注意格式化默认格式应为 exFAT 如果不是请修改！
卷标就是U盘的名字，后期也可以手动修改
PE壁纸如果你想加的话点击右边三个点选择你的图片即可
[可选]包含DOS工具箱
然后点击 立即安装进U盘 泡个泡面或者趁现在去下个Windows镜像~

单系统

分区

O进入PE后
①打开 分区工具DiskGenius （傲梅）分区助手功能一样，但界面有些许不同，教程以分区工具DiskGenius界面制作
②点击 快速分区
③在 当前硬盘 处选择完硬盘后开始设置分区参数
分区表类型 GUID(GPT) MBR
分区数目 2 2
重建主引导记录（MBR） √
保留现有ESP分区
创建新ESP分区 300MB
创建MSR分区 √

GPT和MBR选择一个然后按表填就好，Win10及以上推荐GPT
对齐分区到此扇区数的整数倍 > √ > 4096扇区 优化硬盘性能及寿命，此操作也称为4K对齐4KB对齐

建议装有系统的硬盘仅分2个区，一个用于系统一个用于存放文件及安装软件
分区分配容量大小建议值见下表
硬盘大小 120GB 256GB 512GB 1TB
系统分区 50GB 80GB 120GB 200GB
其他分区 ≈60GB ≈176GB ≈392GB ≈824GB

数值仅供参考，不同硬件会有些许误差纯属正常！
请注意，上表的数值是几乎完全留给系统以及垃圾文件的，所以请尽量不要往C盘安装软件！
如果需要往C盘安装软件请自行扩大容量！或者完全不分区，一个区走天下
④开始分区

系统安装

从桌面上打开 Windows 安装器
注意顶上的系统选择是不是 Windows Vista/7/8/10/11/2008/20XX （默认值）
①选择 Windows 安装文件的位置 > 点击搜索选择ISO镜像文件
选择完毕后会在下面 选项 处出现 系统选择 ，随便选反正有激活工具 图中被挡住了，已打箭头标识
推荐安装专业版或者专业工作站版
选择引导驱动器的位置 > 可以直接点 小箭头 选择，
如果你刚刚分的是 GPT分区 > 选择 绿的分区 EFI System(ESP) ，不要选U盘作死！
如果你刚刚分的是 MBR分区 > 选择分好的C盘即可
顺便在右侧会显示分区兼容状态，如果没有全绿那可能是分区出了问题，尝试重新分下区
②选择安装驱动器的位置 > 可以直接点 小箭头 选择，点击你需要安装系统的分区，例如刚刚分好50GB的C盘
③选择 模式，推荐 Wimboot:WIMCOPY 速度快
④直接确定
⑤系统安装完后会弹出重启与确认窗口，点击重启。
这里的重启有的时候未必会成功，如果没有自动重启请手动重启！

开始向导

推荐在断网的情况下进行开始向导，否则部分内容会加载的比较久 比如拔掉网线？
联网的步骤也大抵一样，如果不想登录微软账号注意底部是否有类似于 继续执行受限设置 的按钮
如果你重启完直接进了PE系统请返回到 引导 部分手动选择启动项为 Windows Boot Manager 如果没有则选择 你的硬盘 ，或者 关机 拔出U盘后 再开机

进阶配置

随缘更新

关闭系统更新

不一定有用，但是必要设置的一部分
Win+R 在运行窗口输入 services.msc 并 确认 打开
双击 进入以下服务 停止 并将 启动类型 改为 禁用

服务名称：BITS
显示名称：Background Intelligent Transfer Service

服务名称：wuauserv
显示名称：Windows Update

禁用粘滞键

连点五次 Shift 会弹出粘滞键菜单
点击 在“轻松使用”键盘设置中禁用此键盘快捷方式 进入设置后自行关闭相关设置

调整虚拟/页面内存

一般不需要调整，电脑自己会分配
右键 桌面或文件管理器里的 此电脑 点 属性
找到 高级系统设置 并点进去
点击 性能 > 设置 ，顶部 高级 ，虚拟内存 > 更改
关闭 自动管理所有驱动器的分页文件大小，选择一个固态硬盘的分区
自定义大小 > 初始大小 为最大值的四分之一差不多，最大值 为 物理内存 的 1.5 至 2倍
随后 确定 应用 可能需要重启电脑，会有提示

1GB = 1024MB，请使用计算器进行计算以得出需要设置的数值
根据测试在电脑自动分配的情况下，32GB物理内存的电脑会自行分配到38GB上下的虚拟内存
开机状态占用为3GB可以将初始值设置为3GB
如果你嫌32GB物理内存需要分配的虚拟内存太大可以参考一下并设置
每台电脑的运行环境大不相同，仅供参考
